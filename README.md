# FP-Bigdata-A3
## Cara start
### 1. Download dataset
https://www.kaggle.com/datasets/tapive/google-play-apps-and-games
### 2. Start producer
```py
python producer.py
```
### 3. Start consumer
```py
python consumer.py
```
### 4. Start spark
```py
feature_extraxtor.py
```
```py
spark_script.py
```
### 5. Jalankan App.py
```py
python app.py
```
